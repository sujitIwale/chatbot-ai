Chat Widget Integration

Add this script to your website to integrate the chat widget:

<script src="http://localhost:3000/embed.js" data-chatbot-id="your-chatbot-id" data-chatbot-name="Support Bot" data-primary-color="#3B82F6" ></script>
Required: Replace your-chatbot-id with your actual chatbot ID.

Configuration Options

  • data-chatbot-id - Your chatbot ID (required)
  • data-chatbot-name - Bot display name (default: "AI Assistant")
  • data-primary-color - Widget color (default: "#3B82F6")
  • data-position - "bottom-right" or "bottom-left" (default: "bottom-right")
  • data-initial-message - First bot message
  • data-api-base-url - API server URL (default: "http://localhost:3001")

Example

<script src="http://localhost:3000/embed.js" data-chatbot-id="demo-bot-123" data-chatbot-name="Help Assistant" data-primary-color="#10B981" data-position="bottom-left" data-initial-message="Hi! How can I help you?" ></script>
